## Important logo rule

The logo wordmark is hardcoded as:

```text
AI Ledger
```

The React component does not expose an editable product-name prop.

## Included files

```text
assets/logo/svg/               Vector logo assets
assets/logo/png/               PNG logo exports
assets/reference/              Selected reference board
components/AILedgerLogo.tsx    Locked React logo component
components/brand-theme.ts      Typed theme values
docs/brand-guide.md            Full identity guidance
docs/codex-handover.md         Codex implementation instructions
examples/example-shell.tsx     Example use
styles/fonts.css               Font imports
styles/theme.css               Theme variables and classes
tokens/design-tokens.json      Machine-readable tokens
tokens/tailwind-theme-snippet.ts Tailwind configuration snippet
```

No font files are included. Install the specified font package through npm.
